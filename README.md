# ブラックジャック

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1
* 必要なものがあれば追記してください（非推奨）

## ゲームの概要
* プレイヤーはディーラよりもカードの合計が21点に近ければ勝利、勝利時チップを得ることが出来る
* カードの合計が21点を超えたら敗北
* 全てのチップを失うとゲームオーバー

## ゲームの遊び方
* 十字キーで掛けるチップの量を設定しEnterキーで選択する
* HキーでHIT・・・手札にカードを1枚加える、SキーでSTAND・・・カードを加えない、DキーでDOUBLEDOWN・・・掛けたチップを倍にして1枚のみカードを引く

## ゲームの実装
### 共通基本機能
* 背景画像とカード(数)の描画
* HIT、STAND、DOUBLEDOWN機能の実装
* 勝敗の設定（ゲームオーバーではない）

### 分担追加機能
* メインメニュー画面 (小山)：特定のボタンを押してゲームを開始するクラス
* ゲームオーバー画面 (郷)：画面を暗くしてゲームオーバーを表示また、コンテニュー処理(ゲームを新たに再開)を作成するクラスまたは条件
* ブラックジャック画面 (内倉)：ブラックジャック発生時にブラックジャックを表示させるクラス
* 掛け金（チップ）(大岡)：現在の所持チップと掛けるチップを表示、勝敗時の配当金の計算をするクラス
* ダブルダウンの実装 (千葉)：ダブルダウンのボタンと機能を実装するクラス

### メモ
* クラス内の変数は，すべて，「get_変数名」という名前のメソッドを介してアクセスするように設計してある
* すべてのクラスに関係する関数は，クラスの外で定義してある
* 現在の案：クイズゲーム、ブラックジャック　（メインメニュー　ゲームオーバー　掛け金　ダブルダウン　フラックジャック時の画面）
